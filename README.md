# Virtualid
Plataforma web y móvil para la identificación virtual y la gestión de acceso. 
Se contienen 4 carpetas en las que cada una cuenta con una parte de la aplicación completa:

1.Landing Page: Página web publicitaria en donde se explica los servicios que ofrece VIRTUALID (Fue desarrollada en Next.js).
2.Aplicación Web: Software que brinda servicios administrativos a clientes o empresas interesadas en la gestión de empleados (Desarrollada en angular).
3.Aplicación Móvil: Software que brinda funnciones a usuarios/empleados vinculados a la empresa que contrata los servicios en la web (Desarrollada en ionic).
4.Backend: Aplicación que contiene los microservicios para gestionar la lógica de negocio, los datos, y la seguridad de todo el conjunto de aplicaciones que forma virtualid. (Desarrollada en Node.js y Express)
